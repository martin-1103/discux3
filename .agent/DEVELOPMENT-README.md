## Development Workflow

### Session Management

1. **Start Session**: `Start my Navigator session`
   - Loads project context from `.agent/`
   - Activates task management system
   - Sets up memory persistence

2. **Create Tasks**: `Create task for [feature]`
   - Generates implementation plans in `.agent/tasks/`
   - Auto-assigns task numbers (TASK-001, TASK-002, etc.)
   - Creates structured documentation

3. **Save Progress**: `Save my Navigator session`
   - Persists session memory for continuity
   - Creates context markers for next session

## Current Features

Based on `package.json` analysis:
- **Authentication**: NextAuth 5.0.0-beta with Prisma adapter
- **Database**: Prisma ORM with PostgreSQL
- **UI**: Radix UI components with Tailwind CSS
- **State**: Zustand for client state management
- **Vector Search**: Qdrant for vector operations
- **Real-time**: Socket.io for websocket connections
- **Forms**: React Hook Form with Zod validation

## Development Standards

### Code Organization
- Follow Next.js App Router conventions
- Use TypeScript strict mode
- Components in `src/components/` organized by feature
- Server actions in `src/lib/actions/`
